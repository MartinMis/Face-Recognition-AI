# Face Recognition AI
> [!NOTE]
> You might have noticed this repository has almost no history. This is intentional as this repository serves as a publicly presentable view of a project originally developed in private repository.
> History has been omitted to prevent leaking of private information.

This project was originally created by the team of 3 people. It implements facial recognition siamese neural network. The network was trained and tested on [LFW - People](https://www.kaggle.com/datasets/atulanandjha/lfwpeople)
dataset. The project compares two different loss function with the aim of identifying the one with better results.
